Setup Checklist
1
Server is running ✓
FastAPI server started on port 8000
2
Expose with ngrok
Run ngrok http 8000 in a new terminal. Copy the https://xxxx.ngrok.io URL.
3
Configure Meta Webhook
Go to Meta Developer Console → your app → WhatsApp → Configuration → set Callback URL to https://xxxx.ngrok.io/webhook and Verify Token to loading...
4
Subscribe to messages webhook
In Meta Console → Webhook Fields → subscribe to messages
5
Add your phone as test recipient
Meta Developer Console → WhatsApp → API Setup → add your phone number as recipient. Then send a message to your test WhatsApp number.
